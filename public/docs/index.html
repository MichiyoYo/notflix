Overview

The web application NotFlix will provide users with access to information about different movies, directors, and genres by offering a set of endpoints to achieve different results.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Available Features

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it‚Äôs featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., ‚ÄúThriller‚Äù)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

Extra Features

  • Allow users to see which actors star in which movies
  • Allow users to view information about different actors
  • Allow users to view more information about different movies, such as the release date and the movie rating
  • Allow users to create a ‚ÄúTo Watch‚Äù list in addition to their ‚ÄúFavorite Movies‚Äù list
